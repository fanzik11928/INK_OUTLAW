INK OUTLAW

2D Rubber-Hose Boss Fight Game

A stylish 1930s cartoon-style boss fight game inspired by Cuphead.

## Quick Start

### Python Version
```bash
pip install pygame
python INK_OUTLAW_Game.py
```

### Browser Version
Open INK_OUTLAW_Standalone.html

Controls:
- A/D - Move
- Space - Jump
- J - Shoot
- K - Parry

## Files
- INK_OUTLAW_Game.py - Standalone Pygame game
- INK_OUTLAW_Standalone.html - Browser version

Made with love for retro animation fans.
